# 屏蔽windows系统更新

  1. 调出运行对话框： +
  2. 输入“service.msc”，停止“windows update”，并禁用；同时在恢复里，改为“无操作”
  3. 返回到运行对话框界面
  4. 输入“gpedit.msc”，“管理模板”→“Windows组件”→“Windows更新”，双击，然后找到“配置自动更新”，设置为“已禁用”
